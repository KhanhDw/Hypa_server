khi mới clone về từ github vui lòng tải các file cấu hình cần thiết cho hệ thống:

1. tải 3 file cho vào ffmpeg\bin
   ffmpeg.exe
   ffplay.exe
   ffprobe.exe
   => cần 3 file này để tải video và audio kèm âm thanh
   link tải: https://1drv.ms/f/c/58f6b1caf07f2f26/IgANhWurlt94TJMwgMAwo68iAfCuGYZRbXhHiV8vTpwT3-E?e=2EQJpO

2. tải nodejs để cho hệ thống dùng javascript biên dịch
   => chọn gói có tên tương tự: node-v25.2.1-win-x64.zip (nếu dùng window)
   => giải nén và đổi tên thành "nodejs" để ở thư mục root, thay thế thư mục nodejs hiện có trong dự án
   link tải: https://nodejs.org/dist/latest/
